## 📌 Tentang Project
Project ini merupakan aplikasi **Point of Sale (POS)** berbasis web yang dikembangkan menggunakan **Laravel** selama kegiatan magang di **PT Lentera Jiwa**.

Aplikasi ini dirancang untuk membantu pengelolaan transaksi penjualan secara efisien dan terstruktur.

---

## ✨ Fitur Utama
- 🔐 Authentication (Login & Register)
- 📦 Manajemen Produk (CRUD)
- 🛒 Transaksi Penjualan
- 📊 Dashboard Statistik
- 👤 Manajemen User
- 🧾 Riwayat Transaksi

---

## 🛠️ Teknologi yang Digunakan
- **Laravel** (Backend Framework)
- **PHP**
- **MySQL**
- **Bootstrap / Template Admin**
- **JavaScript**
